<center><h2>📕 Commit Convention</h2><center>

> 해당 컨벤션을 꼭 지켜주세요.

```
git commit -m "본인 이름: 문제종류 (문제번호) 문제제목"
```
예를 들어, 홍길동이 백준 사이트의 1000 번호의 A+B 문제를 풀었습니다.<br/>
그렇다면 "홍길동: BJ 1000 A+B" 가 됩니다.<br/>

